# ProjetoBancoG4

este projeto foi feito em acompanhamento da aula de C# Foursys

Como está estruturado este projeto: inteiramente orientado a objeto, possui heranças, interfaces e também possui ligação a planilhas .csv possibilitando a gravação e leitura de dados, este projeto tem por finalidade a possibilidade de que o usuário possa realizar um cadastro bancário, onde também estará incluida, a possibilidade de realizar transferencias, depositos e consulta de conta, também foi integrado a criação de conta corrente e poupança, podendo assim, ser realizadas calculos de juros e taxas de rendimentos, também é possivel a realização de cadastros de chaves pix, atravez da integração de interfaces de leitura e gravação para uma planilha excel .csv
